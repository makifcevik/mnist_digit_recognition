# Dataset

The project uses MNIST digits dataset, consisting of 60,000 training images and 10,000 test images.

- Input: 28x28 grayscale images (flattened to 784 features).
- Output: 10 classes (Digits 0-9).

Note: The dataset loader expects the standard binary (`.idx1` & `.idx3`) format.
You can download and paste the dataset files in this directory. (e.g from [Kaggle](https://www.kaggle.com/datasets/hojjatk/mnist-dataset)).
